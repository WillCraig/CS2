Department of Computer Science

You can create a web page in your wasp/hornet account.

The URL is: http://web.cs.kent.edu/~USERID/

Create a public_html folder. Make it readable to everyone with the following Unix commad:
chmod 755 public_html

Create an index.html file in that folder and add content.
